**TLS** (Transport Layer Security) ‚Äî —ç—Ç–æ —Å—Ç–∞–Ω–¥–∞—Ä—Ç–Ω—ã–π –ø—Ä–æ—Ç–æ–∫–æ–ª —à–∏—Ñ—Ä–æ–≤–∞–Ω–∏—è, –∫–æ—Ç–æ—Ä—ã–π –æ–±–µ—Å–ø–µ—á–∏–≤–∞–µ—Ç –±–µ–∑–æ–ø–∞—Å–Ω—É—é –ø–µ—Ä–µ–¥–∞—á—É –¥–∞–Ω–Ω—ã—Ö (–Ω–∞–ø—Ä–∏–º–µ—Ä, HTTPS). –í —Å—Ç–∞–Ω–¥–∞—Ä—Ç–Ω–æ–º TLS –∫–ª–∏–µ–Ω—Ç –ø—Ä–æ–≤–µ—Ä—è–µ—Ç –ø–æ–¥–ª–∏–Ω–Ω–æ—Å—Ç—å —Å–µ—Ä–≤–µ—Ä–∞ –ø–æ –µ–≥–æ —Å–µ—Ä—Ç–∏—Ñ–∏–∫–∞—Ç—É.

**mTLS (Mutual TLS)** ‚Äî —ç—Ç–æ —Ä–∞—Å—à–∏—Ä–µ–Ω–∏–µ TLS, –≥–¥–µ –ø—Ä–æ–∏—Å—Ö–æ–¥–∏—Ç **–≤–∑–∞–∏–º–Ω–∞—è** –∞—É—Ç–µ–Ω—Ç–∏—Ñ–∏–∫–∞—Ü–∏—è. –ù–µ —Ç–æ–ª—å–∫–æ –∫–ª–∏–µ–Ω—Ç –ø—Ä–æ–≤–µ—Ä—è–µ—Ç —Å–µ—Ä–≤–µ—Ä, –Ω–æ –∏ —Å–µ—Ä–≤–µ—Ä –ø—Ä–æ–≤–µ—Ä—è–µ—Ç –∫–ª–∏–µ–Ω—Ç–∞. –û–±–∞ –¥–æ–ª–∂–Ω—ã –ø—Ä–µ–¥–æ—Å—Ç–∞–≤–∏—Ç—å –≤–∞–ª–∏–¥–Ω—ã–µ —Å–µ—Ä—Ç–∏—Ñ–∏–∫–∞—Ç—ã, —á—Ç–æ–±—ã —É—Å—Ç–∞–Ω–æ–≤–∏—Ç—å —Å–æ–µ–¥–∏–Ω–µ–Ω–∏–µ.

**–ö–∞–∫ —ç—Ç–æ —Ä–∞–±–æ—Ç–∞–µ—Ç –≤ Istio:**

2.  **Sidecar-–ø—Ä–æ–∫—Å–∏**: –ö–æ–≥–¥–∞ `order-service` —Ö–æ—á–µ—Ç —Å–≤—è–∑–∞—Ç—å—Å—è —Å `product-service`, `istio-proxy` –≤ –ø–æ–¥–µ `order-service` –ø–µ—Ä–µ—Ö–≤–∞—Ç—ã–≤–∞–µ—Ç –∑–∞–ø—Ä–æ—Å.
3.  **Handshake**: –ü—Ä–æ–∫—Å–∏ `order-service` –∏ `product-service` –∏–Ω–∏—Ü–∏–∏—Ä—É—é—Ç mTLS-—Ä—É–∫–æ–ø–æ–∂–∞—Ç–∏–µ. –û–Ω–∏ –æ–±–º–µ–Ω–∏–≤–∞—é—Ç—Å—è —Å–µ—Ä—Ç–∏—Ñ–∏–∫–∞—Ç–∞–º–∏, –≤—ã–¥–∞–Ω–Ω—ã–º–∏ `istiod`, –∏ –ø—Ä–æ–≤–µ—Ä—è—é—Ç –∏—Ö –ø–æ–¥–ª–∏–Ω–Ω–æ—Å—Ç—å.
4.  **–®–∏—Ñ—Ä–æ–≤–∞–Ω–Ω—ã–π —Ç—É–Ω–Ω–µ–ª—å**: –ü–æ—Å–ª–µ —É—Å–ø–µ—à–Ω–æ–π –ø—Ä–æ–≤–µ—Ä–∫–∏ —Å–æ–∑–¥–∞–µ—Ç—Å—è –∑–∞—à–∏—Ñ—Ä–æ–≤–∞–Ω–Ω—ã–π —Ç—É–Ω–Ω–µ–ª—å, –ø–æ –∫–æ—Ç–æ—Ä–æ–º—É –ø–µ—Ä–µ–¥–∞—é—Ç—Å—è –¥–∞–Ω–Ω—ã–µ.
5.  **–ü—Ä–æ–∑—Ä–∞—á–Ω–æ—Å—Ç—å –¥–ª—è –ø—Ä–∏–ª–æ–∂–µ–Ω–∏—è**: –°–∞–º–æ –ø—Ä–∏–ª–æ–∂–µ–Ω–∏–µ (`order-service`) –Ω–∏—á–µ–≥–æ –Ω–µ –∑–Ω–∞–µ—Ç –æ mTLS. –û–Ω–æ –ø—Ä–æ–¥–æ–ª–∂–∞–µ—Ç –æ—Ç–ø—Ä–∞–≤–ª—è—Ç—å –æ–±—ã—á–Ω—ã–π HTTP-—Ç—Ä–∞—Ñ–∏–∫, –∞ –≤—Å—é —Ä–∞–±–æ—Ç—É –ø–æ —à–∏—Ñ—Ä–æ–≤–∞–Ω–∏—é –∏ –∞—É—Ç–µ–Ω—Ç–∏—Ñ–∏–∫–∞—Ü–∏–∏ –±–µ—Ä–µ—Ç –Ω–∞ —Å–µ–±—è Istio.

## –ú–∞—Å—à—Ç–∞–±–∏—Ä—É–µ–º–æ—Å—Ç—å –º–∏–∫—Ä–æ—Å–µ—Ä–≤–∏—Å–æ–≤ –≤ Kubernetes (HPA)
